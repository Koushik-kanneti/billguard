Product in Action

60 seconds from bill to clarity.

📸 Step 1 — Capture

Select bill type (Hospital / Restaurant / Online Order). Upload a photo or use live camera. App rejects blurry images with real-time blur detection before wasting an API call.

🧠 Step 2 — Analysis

Gemini Vision reads every character. A rule engine built from Indian law checks each charge. Server recalculates math — never trusts the AI's arithmetic.

⚠️ Step 3 — Your Report
⚠️ HIGH RISK — ₹259 overcharged
Service Charge: ₹150 → ₹0 (CCPA 2022)
Diet Coke × 2: ₹160 → ₹80 (MRP violation)
Extra GST: ₹29 → ₹0 (on banned charge)

Each flag: what to say, which helpline to call, what form to file.

🗺️ Step 4 — Scam Map

Business name + address extracted automatically from the bill. Geocoded and pinned. If 10 others flagged the same place — the pin glows red.
